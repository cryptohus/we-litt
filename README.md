## What is We Litt?

We Litt is a mobile-first single-page app for discovering the best nightlife, day parties, brunches, block parties, concerts, and cultural events across 8 major US cities — built specifically for Black and urban culture.

**Cities covered:** Atlanta · Miami · New York City (all 5 boroughs) · Los Angeles · Houston · Chicago · Dallas · Washington DC

## Features

- 🔥 **200+ curated events** — real block parties, cultural festivals, parades, and nightlife
- 📍 **Zip code search** — type any zip (e.g. `11238` for Crown Heights, `77004` for Third Ward Houston)
- 📅 **Date filters** — Today / Tomorrow / This Weekend / This Week / Pick a Date
- ⚙️ **Sort & Filter** — by Popularity, Date, Distance, Price; filter by price range and proximity
- 🏙️ **Browse by City** — city cards with live event counts
- 🗺️ **Map view** — events plotted on an interactive map
- 🖤 **Tier membership** — Free Explorer / Litt+ ($9.99/mo) / Litt Black ($29.99/mo)
- 🎟️ **Promoter dashboard** — submit events with tiered ticketing and 9% We Litt kickback model
- 🤖 **Concierge chatbot** — book tickets, tables, and bottle service via AI assistant
- 📱 **PWA** — installable on iPhone and Android

## Tech

Single HTML file · No framework · No build tools · Deployed via GitHub Pages
